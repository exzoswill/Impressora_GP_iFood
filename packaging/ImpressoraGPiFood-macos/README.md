# ImpressoraGPiFood macOS Packaging

Este diretório contém material para empacotamento do Impressora_GP_iFood como uma aplicação macOS redistribuível.

Este projeto é uma redistribuição/derivação do projeto original:

- Origem: https://github.com/cleitonleonel/Impressora_GP_iFood
- Licença original: conforme repositório de origem (ver arquivo original)

Nós empacotamos e fornecemos scripts para criar um .app e um .dmg, além de instruções para codesign e notarização.

Se você redistribuir este empacotamento, mantenha referência ao repositório original e respeite sua licença.
